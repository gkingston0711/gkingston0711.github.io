TMSys

Thermal Monitoring System

Thermal Monitoring System was designed to be an automated system that used cheap lightweight hardware to track an individuals temperature using Raspberry Pi 3 and MLX06990 thermal camera. Once activated the system records a users temperature at regular intervals and records them. Once enough data is tracked the system keeps track of the users average, and if the temp becomes too high or too low and email is sent with all the recorded data.
